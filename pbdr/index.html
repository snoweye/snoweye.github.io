#### What is this? This is a website about ``Programming with Big Data in R'' and provide several <a href="http://www.r-project.org/" target="_blank">R</a> packages. We develop pbdR packages and focus on analyzing big data by utilizing Statistics methodology. All packages mainly support the <a href="../hpsc/spmd.html">SPMD</a> programming model with special focus on large scale computing clusters. Packages are available in the <a href="./download.html">Download</a> and instruction for <a href="./install.html">Installation</a> is also available. Some basic examples can be found in the <a href="../hpsc/cookbook.html">Cookbook</a> section of <a href="../hpsc/index.html">HPSC</a> website. --- #### Official Website The <a href="http://r-pbd.org/" target="_blank">Programming with Big Data in R</a> (pbdR) has an official website at <a href="http://r-pbd.org/" target="_blank">http://r-pbd.org/</a> . The information remaining here is for internal development, technical discussion, and windows binary download.
