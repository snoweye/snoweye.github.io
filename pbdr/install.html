Install Steps

I share my scripts of building pbdR in xubuntu and virtualbox system. Some of next steps are optional, but I keep them for the record.

  1. Install R-devel: build_R.
  2. Install OpenMPI-1.6.5: build_ompi.
  3. Install HDF5-1.8.11: build_hdf5.
  4. Install NETCDF-4.3.0: build_netcdf.
  5. Install pbdR: build_pbdR.
Note that build_pbdR also assumes some required R packages are installed prior, then download source from pbdR developing website at GitHub. Next I share my script to load pbdR environment.
  • Environment: 00_set_R.
