Run pbdR Packages on Virtual Machine

This section demonstrate how to install Oracle VM VirtualBox on a Windows machine, install pbdR image (1.7GB) to the VirtualBox, and run a quick test of pbdR. The pbdR image contains all installed pbdR packages and environments such as MPI, HDF5, and NetCDF libraries. I assume all files are installed using default settings in most steps. Please click the following subsections:

  2. "> Install pbdR Image.
