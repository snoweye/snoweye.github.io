mpi.universe.size {Rmpi}R Documentation

MPI\_Universe\_size API

Description

mpi.universe.size returns the total number of CPUs available in a cluster. Some MPI implements may not have this MPI call available.

Usage

  mpi.universe.size()

Arguments

None.
