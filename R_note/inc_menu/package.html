And the fileds 'Depends' and 'Suggests' are <br> <ol> <li> The necessary to successfully load must be listed in 'Depends'. <li> To successfully run <b>R CMD check</b> must be listed either in 'Depends' or 'Suggests'. </ol> <br> <li> <font color="#800000"><b>The 'INDEX' file</b></font> <br> <font color="green">Optional</font>. <br> Automatically generated from <b>Rdindex()</b> from package tools. <br><br> <li> <font color="#800000"><b>The 'R' subdirectory</b></font> <br> Contains R code files. <br> The extension '.R' or '.r'. <br> The 'call_c.r' and 'call_f.r' have created in <a href="<? echo "$PHP_SELF?item=R_Call"; ?>">R_Call</a> page and modify to suit the package. <br> The 'zzz.r' for load the package. <br> <font color="red">Put your R code in this subdirectory.</font> <br><br> <li> <font color="#800000"><b>The 'README' file</b></font> <br> <font color="green">Optional</font>. <br> Summary this package. Here, it record the process to build a package. <br><br> <li> <font color="#800000"><b>The 'demo' subdirectory</b></font> <br> <font color="green">Optional</font>. <br> It will running via <b>demo()</b>. <br> The '00Index' contains the summarize for all demo function. <br> The 'RcallC.R' and 'RcallF.R' are the part of 'cal_c.r' and 'call_f.r' have created in <a href="<? echo "$PHP_SELF?item=R_Call"; ?>">R_Call</a> page and modify to suit the package. <br><br> <li> <font color="#800000"><b>The 'man' subdirectory</b></font> <br> Contains documentation files for the objects in the package in <i>R document</i> (Rd) format. <br> The extension '.Rd'. <br> The 'test.c.Rd' and 'test.f.Rd' are the document for function 'test.c' and 'test.f' which described in the 'call_c.r' and 'call_f.r' belong the 'R' subdirectory. <br><br> <li> <font color="#800000"><b>The 'src' subdirectory</b></font> <br> <font color="green">Optional</font>. <br> Contains C, C++, FORTRAN source files and optionally file 'Makevars' or 'Makefile'.<br> The 'call_c.c' and 'call_f.f' have created in <a href="<? echo "$PHP_SELF?item=R_Call"; ?>">R_Call</a> page and modify to suit the package. <br> <font color="red">Put your source code (.c, .f, .cc) in this subdirectory.</font> <br> <br><hr><br> <li> <font color="#800000"><b>Compile & Build & Check</b></font> <br> <b>Linux:</b> <br> <table cellpadding="5"> <tr> <td> &nbsp;&nbsp;&nbsp;&nbsp; <td bgcolor="#dddddd"> <code> SHELL> R CMD build --force ./mypkg <br> SHELL> R CMD check ./mypkg <br> SHELL> R CMD build ./mypkg <br> </code> </table> <br> <ol> <li> 'build' will automatically create the 'INDEX' file if not exist. <li> 'check' will create a subdirectory "mypkg.Rcheck" and simulate the install process and test all function could work. <li> 'build' will create a tar.gz file, the package 'mypkg'. </ol> And the process message may be like this "<a href="./example/package/message.linux.txt" target="_blank">message.linux.txt</a>". <br> <br> <b>Windows:</b> <br> <b> <font color="red"> Require some additional tools for build pacakge or install from source code. <br> <ol> <li> See the page <a href="http://www.stats.ox.ac.uk/pub/Rtools/" target="_blank">Building R for Windows</a>. <li> See also the <a href="<? echo "$PHP_SELF?item=windows"; ?>">Windows</a> page to construction the environment for Windows. </ol> </font></b> <br> After the environment ready, the build process as follow, <br> <ol> <li> Download <a href="./example/package/mypkg_0.1-1.tar.gz">mypkg_0.1-1.tar.gz</a>. <li> Unzip mypkg_0.1-1.tar.gz to C:\WINDOWS\DESKTOP <li> Open MS-DOS command mode. <br> <table cellpadding="5"> <tr> <td bgcolor="#dddddd"> <code> C:\> COPY C:\WINDOWS\DESKTOP\MYPKG C:\PROGRA~1\R\RW1071\SRC\LIBRARY <br> C:\> C:\PROGRA~1\R\SET_BUILD <br> > CD C:\PROGRA~1\R\SRC\GNUWIN32 <br> > make pkg-mypkg <br> </code> </table> <li> "SET_BUILD.BAT" will set the building environment. <li> "make pkg-mypkg" will build all package and install to R library path. </ol> <br> The process message may be like this "<a href="./example/package/message.windows.txt" target="_blank">message.windows.txt</a>". <br> The complete package will be at "C:\PROGRA~1\R\RW1071\LIBRARY\MYPKG", you can zip the whole directory to <a href="./example/package/mypkg_0.1-1.zip">mypkg_0.1-1.zip</a>. <br><br> <li> <font color="#800000"><b>Install</b></font> <br> See <a href="<?
